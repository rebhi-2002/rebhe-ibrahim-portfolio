## About This Project

This repository contains the code for my personal portfolio website. It was built from scratch with a mobile-first approach and a strong focus on performance, animations, and creating an exceptional user experience that reflects my professional brand.

### Key Features:

* **Interactive UI:** Smooth animations and transitions powered by Framer Motion.
* **Case Studies:** A dedicated section to showcase my best work in detail.
* **Dynamic Toolbox:** A comprehensive and filterable list of my technical skills.
* **Responsive Design:** Fully responsive and optimized for all screen sizes.

## Built With

This project was built using the following modern technologies:

* **[React.js](https://react.dev/):** A JavaScript library for building user interfaces.
* **[TypeScript](https://www.typescriptlang.org/):** For strong typing and a better developer experience.
* **[Tailwind CSS](https://tailwindcss.com/):** A utility-first CSS framework for rapid UI development.
* **[Framer Motion](https://www.framer.com/motion/):** A production-ready motion library for React.
* **[Vite](https://vitejs.dev/):** A modern, fast build tool for front-end development.
* **[Vercel](https://vercel.com/):** For deployment and hosting.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js (v18 or later)
* npm or yarn

### Installation

1.  Clone the repo
    ```sh
    git clone [https://github.com/rebhi-2002/rebhe-ibrahim-portfolio.git](https://github.com/rebhi-2002/rebhe-ibrahim-portfolio.git)
    ```
2.  Navigate to the project directory
    ```sh
    cd rebhe-ibrahim-portfolio
    ```
3.  Install NPM packages
    ```sh
    npm install
    ```
4.  Start the development server
    ```sh
    npm run dev
    ```
